Metodología

Cómo se construye el score.

Cada equipo recibe hasta 100 puntos repartidos en cinco métricas extraídas de Bloomberg PORT, convertidas a puntaje por percentil continuo: el mejor de cada métrica se lleva el peso completo, el peor se lleva 0, y el resto se interpola linealmente. IR, Retorno en Exceso o Sharpe negativos anotan 0 en esa métrica, sin excepción.
